# What is it?
Nexify is a course platform for developers, where you can create your own course/tutorial, and make it visible for everyone.

# Installation
```
npm i
npm start
```

# Documentation
In /docs. Use [mermaid](https://mermaid-js.github.io/mermaid/#/) to read the diagram
